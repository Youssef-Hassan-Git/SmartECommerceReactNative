# 🛍️ Smart E-Commerce (React Native)

A modern, cross-platform e-commerce mobile application built with **React Native**, **Expo**, **TypeScript**, **Firebase**, and **Redux Toolkit**. The application delivers a complete shopping experience, including authentication, product browsing, favorites, shopping cart, checkout, order history, and user profile management.

---

## 📱 Features

### 🔐 Authentication

* User Registration
* User Login
* Secure Firebase Authentication
* Logout Functionality

### 🏠 Home

* Browse Products
* Responsive Product Grid


### 🛒 Shopping Cart

* Add Products to Cart
* Remove Products
* Increase & Decrease Quantity
* Automatic Total Price Calculation
* Cart Persistence using Redux Persist

### 💳 Checkout

* Order Summary
* Shipping Information
* Checkout Confirmation
* Store Orders in Firebase Firestore

### 📦 Orders

* View Previous Orders
* Display Purchased Items
* Order History Retrieved from Firestore

### 👤 Profile

* Display User Information
* Localization (8 languages added)
* Logout

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Youssef-Hassan-Git/SmartECommerceReactNative.git
```

Navigate to the project:

```bash
cd SmartECommerceReactNative
```

Install dependencies:

```bash
npm install
```

Start the Expo development server:

```bash
npx expo start
```

Run on Android:

```bash
npx expo run:android
```

Run on iOS (macOS only):

```bash
npx expo run:ios
```
